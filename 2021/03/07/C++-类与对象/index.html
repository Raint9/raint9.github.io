一.explicit,初始化列表

隐式转换与 explicit

关键词explicit可以使函数避免进行隐式转换
例如Time time=5; 对象time是不会等于5的，其中有隐式转换，加上explicit可以禁止掉。
一般对于单参数的构造函数，都声明为explicit，除非有特别原因。

二.inline,const,mutable,this,static

在类定义中实现成员函数inline

在类定义中实现函数inline：类内的成员实现其实也叫类内的成员函数定义

成员函数末尾的const

const : 常量，在成员函数后面加上一个const。不但要在函数声明中增加const，定义中也要。
作用 : 告诉系统，这个成员函数不会修改该对象里任何成员变量的值等等。
tip:假如 const Time time; 那么整个对象里的成员函数后面只要没有const，都不可以调用
