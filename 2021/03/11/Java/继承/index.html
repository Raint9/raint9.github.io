技巧

1.将公共操作和字段放在超类中

2.不要使用protected字段

3.使用继承来实现“is-a”关系

4.除非所有继承的方法都有意义，否则不要使用继承

5.覆盖方法时，不要改变预期的行为

6.尽量使用多态而不要使用类型信息

7.不要滥用反射
